## Customization

### 1. Customize Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --accent-cyan: #00ffff;      /* Change primary accent color */
    --accent-magenta: #ff00ff;   /* Change secondary accent color */
    /* ... other variables ... */
}
```
